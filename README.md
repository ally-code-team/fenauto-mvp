# IA ABL Oficial

Sistema de Inteligência Artificial do Grupo ABL e UniSantaCruz usando OpenAI Realtime API.

## Funcionalidades

- 🎯 **Dois Contextos Empresariais**: Grupo ABL (tecnologia) e UniSantaCruz (educação)
- 🎙️ **Conversas por Voz**: Tecnologia OpenAI Realtime API
- 💰 **Rastreamento de Custos**: Monitoramento em tempo real dos gastos
- 🌐 **Interface em Português**: Sistema totalmente traduzido
- 📊 **Análise de Sessões**: Custos individuais por sessão e totais

## Tecnologias

- Next.js 15.3.3
- React 19
- OpenAI Realtime API
- TypeScript
- Tailwind CSS

## Ambientes

- **Desenvolvimento**: http://localhost:3000
- **Produção**: https://iaabl.abnr.dev

## Desenvolvimento

```bash
npm install
npm run dev
```

## Produção

```bash
npm run build
npm run start:prod
```

## Configuração

Configure as variáveis de ambiente no arquivo `.env.local`:

```env
OPENAI_API_KEY=your_api_key_here
```
